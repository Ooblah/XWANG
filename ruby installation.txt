Steps to install RUBy:

Windows Installation

Steps 1: Download Ruby http://dl.bintray.com/oneclick/rubyinstaller/rubyinstaller-1.9.3-p484.exe?direct and run exe.

 

Step 2:
Confirm ruby was installed successfully by typing the following command in your terminal:

C:\Users\ssamrai>ruby -v
ruby 1.9.3p484 (2013-11-22) [i386-mingw32]

Confirm Gems is installed correctly

C:\Users\ssamrai>gem -v
1.8.28

Step 3: 
Download PhantomJS from http://phantomjs.org/download.html, please download the zip and place contents in C:\Ruby193\bin (This path will vary depending on where you've installed ruby)

Step 4:
Execute the following commands from the terminal.
      1. gem install selenium-webdriver --no-ri --no-rdoc
      2. gem install cucumber --no-rdoc --no-ri
      3. gem install watir --no-rdoc --no-ri
